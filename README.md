SmartRecipe Hub is a personalized, health-focused recipe management web application designed to simplify cooking for users with dietary restrictions, allergies, and specific food preferences. The platform combines smart filtering, AI-driven features, and meal planning tools to deliver a modern and inclusive culinary experience.
